Banner

The banner requires a url. The recommended image size is 561x300, but it can be whatever you'd like. You must host the image on one of the accepted image hosts in order to comply with the CSP.

Accepted Image Hosts/CDNs

  1. Discord
  2. Twitter
  3. Cloudflare CDN
  4. neet.host

Main CSS

This is the most important. You'll see this everywhere and it defines your experience.

Background

Background is the same throughout every page of PulseRoot, making it the most prominent of all. You define this with the body element.

body {
	background-color: (Hex, RGB, RGBA, etc)
}

Navbar Basics

The navbar is arguably the most complex to work with in terms of Main CSS. There are three parts to the navbar: The main color, the dropdown color, and the hover color. This part of the tutorial will not walk you through the mobile navbar.

ul.navbar {
	background-color: CSS Color; /* The main color of the navbar */
	color: CSS Color; /* Optional - Change the color of the navbar text, defaults to white */
}

.navbar li a:hover, .dropdown:hover .dropbtn {
	background-color: CSS Color; /* Hover coloring, usually best to do rgba(xxx,xxx,xxx,0.05) */
}

.dropdown-content {
	background-color: CSS Color; /* Background color of the Info/Account dropdown items */
}

Links

Links are incredibly easy, but I felt it unnecessary to group them with something else.

a {
	color: CSS Color; /* Change the color links */
}

Input/Buttons

You'll see these scattered from place to place.

input, button, textarea {
	background-color: CSS Color; /* Change the background color */
	color: CSS Color; /* Change the text color, best for matching the background */
	border: ?px solid color; /* Border styling - best to keep solid */
}

Chat CSS

This is for the Global Chat and private messaging.

Messages

This is what messages look like.

#messages li {
	background-color: CSS Color; /* The background color of first/odd messages */
}

#messages li:nth-child(even) {
	background-color: CSS Color; /* Background of second/even messages */
}

Additional/Optional CSS

There is a lot of additional css you may wish to use to enhance your experience and keep your fitting theme, or even add on to it. You can do a lot more than this, but these are just some examples. If you like looking at the source, take a look at the default.css and chat.css.

.display {
	color: CSS Color; /* This will change the color of display names in chat. If left blank, it will use user name colors. */
}

.msg {
	color: CSS Color; /* This will change the color of text in chat messages. Gets overrided by !!color command */
}

::selection, ::-moz-selection {
	/* ::selection is when you click and drag on text. It makes it look fancy! Ooh! */
	background: CSS Color; /* Background color */
	color: CSS Color; /* Text color */
}

hr {
	border-color: CSS Color; /* Changes the color of the <hr/> tags */
}

img {
	border-radius: 0; /* This will remove rounding on images, including profile pictures */
}

.logotitle {
	/* This changes what the word PulseRoot looks like in navbars */
	font-weight: bolder; /* It's best to keep this as bold, but you may find use */
	color: CSS Color; /* Text color */
}

pre {
	/* This changes what code blocks look like. In fact, you're looking at a code block right now! */
	font-family: "Ubuntu Mono", "Consolas", monospace; /* Change the font, best to use a monospaced font */
}
